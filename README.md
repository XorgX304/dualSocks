# dualSocks
CLI socks5 proxy chainer in C#

This is a work in progress. Seems to be reasonably stable & very fast!

# Usage
dsocks2 [user:pass@]proxy.com[:1080] [proxy2 proxy3 ...]

Listens on all interfaces on port 1080 by default
